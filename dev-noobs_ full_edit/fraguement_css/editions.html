Où doit-on écrire

La balise <script >

En HTML, le code JavaScript doit être inserer dans les balises <script > et </script >

Exemples

<script >

document.getElementByTagNames("div").innerHtml = "j'appends à coder en JAVASCRIPT"

</script >

NB: Les vielles versions de javascript utilisées l'attribut <script type = "javascript" > . Cette attribut n'est plus utilisé dans les versions recentes.

Emplacement du JavaScript

Le JavaScript peut être placé soit directement dans la page html entre les balise <head> ou<body> , ou soit dans un ficher externe.
