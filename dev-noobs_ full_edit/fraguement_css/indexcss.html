Pourquoi doit-on apprendre le CSS ?

CSS est l'un des 3 langages web que tous les developpeurs devraient apprendre

1. HTML pour definir le contenu des pages web

2. CSS pour la mise en forme des pages web

3. JAVASCRIPT pour rendre les pages web dynamiques

Les pages Web sont stylés grace à CSS. il est important de le metrisée, parce que maintenant tout repose sur le design donc CSS fait l'objet. Il est incontournable voire indispensable concernant le developpement web. Ils existe aussi des frameworks pour CSS dont le plus connu est Bootstrap.
