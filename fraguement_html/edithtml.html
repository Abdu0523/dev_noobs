Html Introduction

Qu'est-ce que le HTML?

HTML est le langage de balisage standard pour la création de pages Web.

  • HTML signifie Hyper Text Markup Language
  • HTML décrit la structure d'une page Web
  • HTML se compose d'une série d'éléments
  • Les éléments HTML indiquent au navigateur comment afficher le contenu
  • Les éléments HTML sont représentés par des balises
  • Les balises HTML étiquettent des éléments de contenu tels que "en-tête", "paragraphe", "tableau", etc.
  • Les navigateurs n'affichent pas les balises HTML, mais les utilisent pour restituer le contenu de la page
